Newspaper PDFs are saved in this folder.

WHY? Because current news reading techniques in major Nepali News websites are broken.

Folder structure
* combined/<newspaper>/<date>.pdf
* single/<newspaper>/<date>/p[1..16].pdf

The code for generating PDF is available at https://github.com/bibstha/NepaliNewsPdfGenerator
